# FonlaBeni 🚀

🚧 **Durum:** *Bu proje şu an aktif planlama ve geliştirme aşamasındadır.*

FonlaBeni, yenilikçi fikirlere sahip girişimcilerle, bu projelere destek olmak isteyen yatırımcıları ve toplulukları bir araya getirmeyi hedefleyen bir kitle fonlama (crowdfunding) platformudur. 

## 🎯 Proje Hedefleri
- Kullanıcıların kolayca proje oluşturup yayınlayabilmesi.
- Destekçilerin ilgilerini çeken projelere güvenli bir şekilde fon sağlayabilmesi.
- Ölçeklenebilir, bakımı kolay ve sağlam bir arka uç (backend) mimarisi sunmak.

## 🛠️ Kullanılacak Teknolojiler
Projenin geliştirilmesinde aşağıdaki teknoloji yığınından (tech stack) faydalanılması planlanmaktadır:

* **Backend:** C#, .NET Core MVC
* **Veritabanı:** PostgreSQL
* **Mimari:** Clean Architecture
